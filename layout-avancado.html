Layout Responsivo Perfeito

Este layout mantém espaçamento proporcional de ~15cm visuais, cresce automaticamente quando toca as laterais, e preserva proporções perfeitas em qualquer tela.

Características Técnicas

✓ Container: 100vw (encosta nas laterais)
✓ Espaçamento: clamp(8vh, 12vh, 15vh)
✓ Altura: min-height baseado em viewport
✓ Imagem: object-fit: contain (sem distorção)
✓ Responsivo: funciona em todas as telas
✓ Moderno: CSS Grid + Flexbox + clamp()
